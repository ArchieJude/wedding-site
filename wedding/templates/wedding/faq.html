{% trans "The Day" %}

{% if LANG == "zh-hant" %} 我應該幾點到場？

晚間5點開放入場。請於17:15前就座，以便準時開始敬茶儀式。整個晚宴預計至21:00結束。

{% elif LANG == "mn" %} Хэдэн цагт ирэх хэрэгтэй вэ?

Хаалга 17:00 цагт нээгдэнэ. Цайны ёслолыг цагтаа эхлүүлэхийн тулд 17:15 цагаас өмнө сууна уу. Оройн баяр ойролцоогоор 21:00 цагт дуусна.

{% else %} What time should I arrive?

Doors open at 5:00 PM. Please arrive by 5:15 PM so we can start the tea ceremony on time. The evening runs until approximately 9:00 PM.

{% endif %}
{% if LANG == "zh-hant" %} 婚禮的流程是什麼？

晚宴以傳統中式敬茶儀式開場，接著進行西式婚禮誓言，最後在同一個廳享用慶祝晚宴——整個晚上輕鬆流動，而非正式的場次安排。

{% elif LANG == "mn" %} Баярын хөтөлбөр ямар байх вэ?

Оройн баяр уламжлалт Хятадын цайны ёслол-оор эхлэх бөгөөд дараа нь баруун хэв маягийн тангарга өргөх ёслол, тэгэхдээ ижил танхимд баярын зоог барина. Энэ бол хатуу хөтөлбөртэй биш дотно орой байх болно.

{% else %} What is the format of the ceremony?

The evening begins with a traditional Chinese tea ceremony, followed by Western-style vows, then a celebratory dinner — all in the same room. It's an intimate, flowing evening rather than a formal seated programme.

{% endif %}
{% if LANG == "zh-hant" %} 晚宴提供什麼餐點？

我們計劃在北投文物館享用12道式日式懷石料理，並附高山茶服務。如有嚴重過敏或特殊飲食需求，請在RSVP中說明。

{% elif LANG == "mn" %} Хоолны цэс ямар байх вэ?

Бэйтоу музейд 12 дамжлагат Японы кайсэки хооллох төлөвлөгоотой бөгөөд уулын цай хамт орно. Хоолны хязгаарлалт байвал RSVP-д бичнэ үү — хүнд харшил заавал дурдаарай.

{% else %} What's on the menu?

We're planning a 12-course Japanese kaiseki dinner at Beitou Museum, with high-mountain tea service. Flag dietary restrictions in your RSVP — please mention any severe allergies.

{% endif %}
{% if LANG == "zh-hant" %} 場地附近有停車位嗎？

北投文物館附近停車位有限。強烈建議搭乘捷運轉計程車前往（新北投站下車後，再搭車上山），避免停車困擾。

{% elif LANG == "mn" %} Байрлалд зогсоол байна уу?

Бэйтоу музейн ойролцоо хязгаарлагдмал зогсоол байдаг. Орой болохоор зогсоол олоход хэцүү байж болох тул MRT + такси-аар ирэхийг зөвлөж байна (Синьбэйтоу буудал, тэгэхдээ толгодыг дагуу такси).

{% else %} Is there parking at the venue?

Limited parking near Beitou Museum. We strongly recommend arriving by MRT + taxi (Xinbeitou Station, then a short ride up the hill) — parking in the evening can be tricky.

{% endif %}

{% trans "Dress Code" %}

{% if LANG == "zh-hant" %} 服裝有什麼要求？

時尚雞尾酒服裝為主要風格——優雅而舒適。場地是美麗的日式歷史建築，穿著節慶感、略為正式的衣著最為合適。無需正式晚禮服或燕尾服，但請避免過於休閒的穿著。

{% elif LANG == "mn" %} Хувцасны дүрэм ямар байх вэ?

Ухаалаг / коктейлийн хувцас — нарийн боловч тав тухтай хувцас. Газар нь гоёмсог түүхэн Японы барилга тул баярын уур амьсгалтай, чимэглэсэн хувцас тохиромжтой. Хатуу хачир эсвэл оёдол шаардлагагүй, гэхдээ хэт энгийн хувцаснаас зайлсхийгэерэй.

{% else %} What should I wear?

Smart / cocktail attire — elegant but comfortable. The venue is a beautiful historic Japanese building, so something festive and a little polished is perfect. No need for a ballgown or tux, but avoid overly casual.

{% endif %}
{% if LANG == "zh-hant" %} 有哪些顏色需要避免？

依照中式婚禮傳統，賓客通常避免穿著全白或全黑。歡迎穿著色彩鮮豔或喜慶的服裝！

{% elif LANG == "mn" %} Ямар өнгөнөөс зайлсхийх хэрэгтэй вэ?

Хятадын хурмын уламжлалаар зочид цэвэр цагаан эсвэл цэвэр хар хувцас өмсөхгүй байх нь зохистой. Өнгөлөг, баярын хувцас тавтай морил!

{% else %} Any colours to avoid?

Traditionally at Chinese weddings, guests avoid wearing all white or all black. Colour is celebrated — feel free to wear something bold or festive.

{% endif %}
{% if LANG == "zh-hant" %} 鞋子有什麼建議？

場地有傳統木地板，外部也有一小段上坡路。建議穿著舒適跟鞋或平底鞋，避免細高跟鞋。

{% elif LANG == "mn" %} Гутлын талаар зөвлөмж?

Байрлалд уламжлалт модон шал байгаа бөгөөд гадна талд бага зэрэг өгсүүр зам бий. Тохиромжтой өсгий эсвэл хавтгай гутал зөвлөж байна, нимгэн өндөр өсгийнөөс зайлсхийгэерэй.

{% else %} What about shoes?

The venue has traditional wooden floors and a short uphill path outside. Comfortable heels or flats are recommended over stilettos.

{% endif %}

{% trans "Guests & Kids" %}

{% if LANG == "zh-hant" %} 我可以攜伴出席嗎？

我們希望維持約40–50位親友的溫馨規模。若您希望攜伴，請在RSVP中填寫出席人數。如有疑問，歡迎直接與我們聯絡。

{% elif LANG == "mn" %} Хамтрагчаа дагуулж болох уу?

Бид 40–50 хүний жижиг дотно баяр хийхээр төлөвлөж байна. Хамтрагч дагуулах бол RSVP-д хэдэн хүн ирэхийг дурдна уу. Асуулт байвал бидэнтэй шууд холбогдоорой.

{% else %} Can I bring a plus-one?

We're keeping the celebration intimate at around 40–50 guests. If you'd like to bring a partner, note the number in your party on the RSVP. Reach out directly if you have questions.

{% endif %}
{% if LANG == "zh-hant" %} 可以帶小孩嗎？

我們非常喜歡小朋友，但這是一場成人晚宴。懇請安排小朋友在家留守，讓大家輕鬆享受這個美好的夜晚。感謝您的理解！

{% elif LANG == "mn" %} Хүүхдүүдийг дагуулж болох уу?

Бид хүүхдүүдийг маш их хайрладаг ч энэ нь насанд хүрэгчдийн оройн баяр юм. Та хүүхдүүдийгээ гэртээ үлдээж, бүгд тайван баярлах боломж олгоорой. Ойлголцсонд баярлалаа!

{% else %} Are children welcome?

We love kids, but this is an evening adult dinner event. We'd ask that little ones stay home so everyone can relax and enjoy the evening. Thank you for understanding!

{% endif %}
